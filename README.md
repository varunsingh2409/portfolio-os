# Varun's Cybersecurity Portfolio OS
An interactive, high-fidelity Web OS environment acting as a professional cybersecurity and tech portfolio workspace. 

## Features
- **Window Management System**: Fully draggable, stackable, context-aware window interface.
- **Dynamic Spotlight Search**: Launch apps and search skills universally using `Ctrl + K` or the Nav search button.
- **Live Diagnostics**: Desktop widgets mimicking live hardware metrics.
- **Enterprise Theming**: 3 premium themes (Executive Dark, Clean Light, Midnight Monochrome) dynamically remapping CSS context logic.
- **Responsive Architecture**: Graceful fallback transforming the "Desktop OS" interface into a "Mobile App" style perfectly fluid home screen.

## Local Development
Run the site locally using Node:
```powershell
npm start
```
The portfolio will be available at `http://127.0.0.1:4173` or `http://localhost:8080`.

## Deployment
This project is built using strictly **Vanilla HTML, CSS, and JS**. There are no heavy frameworks, npm dependencies, or build scripts needed for production!

To deploy this site and make it live globally for free:
1. Push this entire folder to a **GitHub Repository**.
2. Go to **Settings > Pages** in your GitHub Repo, and set the source branch to `main`.
3. (Optional) Alternative: Link your GitHub repository to **Vercel** or **Netlify**. Both will instantly deploy the site the second you push new code!
